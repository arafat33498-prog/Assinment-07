# 🍃 Keen Keeper

**Keen Keeper** হলো একটি আধুনিক এবং লাক্সারি ড্যাশবোর্ড অ্যাপ্লিকেশন, যা আপনার ব্যক্তিগত সম্পর্কগুলোকে গুছিয়ে রাখতে এবং সুন্দরভাবে ম্যানেজ করতে সাহায্য করে।

---

## 🚀 ব্যবহৃত টেকনোলজি (Tech Stack)

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Deployment:** Netlify / Vercel
- **Version Control:** Git

---

## ✨ মূল বৈশিষ্ট্যসমূহ (Key Features)

1. **Visual Relationship Dashboard:** কার্ড-বেসড ইন্টারফেসের মাধ্যমে এক নজরে সব বন্ধুদের তালিকা এবং স্ট্যাটাস দেখা যায়।
2. **Dynamic Active Navigation:** একটি বুদ্ধিমান ন্যাভবার যা স্বয়ংক্রিয়ভাবে আপনার বর্তমান পেজ হাইলাইট করে।
3. **Premium Minimalist UI:** সফট-ব্লার শ্যাডো এবং কাস্টম থিমের সমন্বয়ে তৈরি একটি লাক্সারি ইউজার ইন্টারফেস।
